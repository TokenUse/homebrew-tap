# TokenUse Homebrew Tap

## Install

```bash
brew tap tokenuse/tap
brew install tokenuse
```

## Upgrade

```bash
brew upgrade tokenuse
```

## Uninstall

```bash
brew uninstall tokenuse
brew untap tokenuse/tap
```
